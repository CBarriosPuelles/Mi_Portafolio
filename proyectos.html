Sistema de Control de Ventas

Esta aplicación fue creada con el fin de llevar el control de las ventas diarias de mi negocio y mantener el stock actualizado, entregándome los totales de los productos vendidos.

Este proyecto fue desarrollado en Visual Studio C#, además de estar enlazado a una base de datos de Mysql.

Sistema de Acreditación

Esta aplicación fue creada con el fin de llevar el control de la asistencia en diferentes congresos médicos y del Banco Central, además de imprimir la credencial de los diferentes asistentes, entregaba la estadística diaria de cantidad de asitentes a las diferentes charlas.

Este proyecto fue desarrollado en Visual Studio C#, además de estar enlazado a una base de datos de Mysql.
